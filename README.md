# VERA Jewelry House — Website

เว็บไซต์สำหรับ VERA Jewelry House สร้างด้วย HTML, CSS และ JavaScript ล้วนๆ ไม่ต้องพึ่ง framework ภายนอก

## วิธีใช้งาน

เปิดไฟล์ `vera_jewelry_house_website.html` ในเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไรเพิ่มเติม

## วิธีอัปโหลดขึ้น GitHub Pages

1. สร้าง repository ใหม่บน GitHub
2. อัปโหลดไฟล์ `vera_jewelry_house_website.html` และ `README.md`
3. ไปที่ **Settings → Pages**
4. เลือก Source: **Deploy from a branch** → branch `main` → folder `/ (root)`
5. บันทึก แล้วรอสักครู่ — เว็บจะพร้อมใช้งานที่ `https://<username>.github.io/<repo-name>/vera_jewelry_house_website.html`

## โครงสร้างไฟล์

```
.
├── vera_jewelry_house_website.html   # หน้าเว็บหลัก (all-in-one)
└── README.md
```

## เทคโนโลยีที่ใช้

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Cormorant Garamond, DM Sans)
- ไม่มี dependency ภายนอก
